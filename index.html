Общая информация

Добро пожаловать на сайт о формирующем оценивании для применения у младших школьников! Здесь вы найдете инструкции, приёмы, шаблоны и рекомендации для эффективного использования формирующего оценивания на уроках.

Материалы могут помогать учителям и студентам педагогических направлений организовывать учебный процесс более эффективно.

Сайт был выполнен в рамках выпускной квалификационной работы по теме «Развитие регулятивных УУД младших школьников в условиях применения формирующего оценивания в процессе обучения».

Поволжская академия образования и искусств имени Святителя Алексия, митрополита Московского, г. Тольятти, Россия.

Страница «О нас» предназначена для ознакомления с данным сайтом.

Страница «Инструкция» содержит информацию о работе с шаблонами и приёмами на данном сайте.

Страница «Приёмы» содержит детальную информацию с приёмами формирующего оценивания, шаблоны которых можно использовать на данном сайте.

Страница «Шаблоны» позволяет использовать шаблоны приёмов формирующего оценивания, дополняя планы уроков для младших школьников.
