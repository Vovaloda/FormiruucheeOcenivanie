Общая информация

Добро пожаловать на сайт формирующего оценивания для применения у младших школьников! Здесь вы найдете инструкции, методики, шаблоны и рекомендации для эффективного использования формирующего оценивания на уроках.

Материалы помогают учителям и педагогам организовывать учебный процесс более эффективно.

Сайт был выполнен в рамках выпускной квалификационной работы по теме "Развитие регулятивных УУД младших школьников в условиях применения формирующего оценивания в процессе обучения".

Страница "О нас" предназначена для ознакомления с данным сайтом.

Страница "Инструкция" содержит информацию о работе с шаблонами.

Страница "Методики" содержит детальную информацию с методиками формирующего оценивания, шаблоны которых можно использовать на данном сайте.

Страница "Шаблоны" позволяет дополнять планы уроков с методиками формирующего оценивания для младших школьников.
